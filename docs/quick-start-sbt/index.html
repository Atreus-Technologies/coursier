sbt plugin

Enable the sbt plugin by adding

resolvers += Resolver.sonatypeRepo("snapshots")
addSbtPlugin("io.get-coursier" % "sbt-coursier" % "0.1.0-mdoc-SNAPSHOT")

either

  • to the project/plugins.sbt file of an sbt project, or
  • to ~/.sbt/1.0/plugins/build.sbt (globally, not recommended).

See the dedicated page for more details.

The sbt plugin only supports sbt 1.x. See the 1.0.x versions of coursier for sbt 0.13.x support.
